## 📊 Project Overview
This project showcases a **Power BI dashboard** built on Amazon Prime streaming data.  
It highlights trends in user engagement, content performance, and subscription growth, demonstrating how data visualization can uncover insights in the OTT (over-the-top) streaming industry.

👉 *Note: This is a basic practice project for visualization using Power BI.*

## 📂 Files
- `amazon-prime.pbix` – Power BI dashboard file  
- `dataset/amazon_prime_data.csv` – Dataset used for analysis  
- `screenshots/` – Preview images of the dashboard  

## 🚀 How to Use
1. Download `amazon-prime.pbix`.  
2. Open in **Power BI Desktop**.  
3. Use slicers and filters to explore the data interactively.  

## 🔍 Key Insights
Movies dominate the platform, making up 80.8% of all content, while TV shows account for only 19.2%.
Drama is the most popular genre with 959 titles, followed by Comedy and Drama-Suspense.
Content growth accelerated after 2015, showing Amazon Prime Video's rapid expansion in recent years.

## 📖 Dataset
The dataset is simulated/public and structured to resemble Amazon Prime streaming data.  
Columns include:  
-	show_id	
-	type	
-	title	
-	director	cast	
-	country	
-	date_added	
-	release_year	
-	rating	
-	duration	
-	listed_in	
-	description
