Data Visualization Using Power BI

📊 Overview
This repository showcases multiple **Power BI dashboards** built on diverse datasets.  
Each project demonstrates skills in **data cleaning, modeling, visualization, and storytelling**.

## 📂 Projects
- [Zepto Dashboard](amazon-dashboard/) – Delivery analytics and customer insights  
- [Amazon Dashboard](amazon-dashboard/) – Sales trends, category performance, and customer behavior  
- [Sales Analysis](sales-dashboard/) – Revenue and regional breakdowns  
- [Finance Dashboard](finance-dashboard/) – Portfolio performance and budget vs. actuals  

## 🚀 How to Use
1. Download the `.pbix` file from the project folder.  
2. Open it in **Power BI Desktop**.  
3. Explore the visuals and interact with filters/slicers.  

## 📸 Screenshots
Screenshots of dashboards are included in each project folder for quick preview.

## 📖 Notes
- Datasets are either public or simulated for demonstration.  
- Each project folder contains its own README with details and insights.
